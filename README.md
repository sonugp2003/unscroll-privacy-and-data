# Unscroll – Privacy & Data

This repository hosts the **Privacy & Data** information page for the Unscroll Android app.

- **App name:** Unscroll  
- **Purpose:** Digital wellness app to reduce short-form video addiction by blocking Shorts, Reels, TikTok, etc. using Android’s Accessibility Service.  
- **File:** `index.html` – a static, mobile-friendly page explaining:
  - What data is collected (via AdMob)
  - What is stored locally on the device
  - What is NOT collected
  - How Accessibility Service is used
  - User controls (reset, opt-out, disable, uninstall)
  - Contact details: `un0scroll@gmail.com`
